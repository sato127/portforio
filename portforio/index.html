WebSite

個人で作成したwebサイトの制作事例です。どちらも架空の企業のものです。テキストエディタはAtomを使用しております。

作品タイトル：Matushima Hotel YURAN GARDEN(架空の企業)
Design / Cording
制作期間：1ヶ月
URL： http://sato127.php.xdomain.jp/yuran/

コメント：
宮城県松島市にあるリゾートホテルを想定して作成した10ページのwebサイトです。html、css、JavaScript、jquery、PHPを使用しており、ブレイクポイント768px未満の画面幅にてレスポンシブに対応しております。
ホテルそのもののイメージとして、松島にあまり見られないリゾートタイプのものを想定しております。20代～40代の松島について伝統的で格式高い、ある種古めかしいというイメージを持っている層の男女の利用を想定し、ホテルの爽やかさ、居心地の良さを感じられるよう白を基調としたシンプルな色でレイアウトを行いました。
周辺情報は松島市公式ホームページやその他施設ホームページへとリンクしていますが、ロゴは自主製作、ページ内画像は著作権フリーの物を利用しております。

作品タイトル：遊食ダイニング 真魚板(架空の企業)
Design / Cording / WordPress
制作期間：1ヶ月
URL： http://sato127.wp.xdomain.jp

コメント：
宮城県仙台市、仙台駅周辺にある飲食店を想定して作成した7ページのwebサイトです。html、css、JavaScript、jquery使用に加え、WordPress化してあります。またブレイクポイント768px未満の画面幅にてレスポンシブに対応しております。
仙台駅周辺に実際にありそうな飲食店のwebサイトをコンセプトに作成しており、和を連想させるような色、画像でレイアウトしました。
ロゴはクラウドソーシングにて作成したものを再利用し、ページ内画像は著作権フリーの物を利用しております。

Skill

webスクールで学習したことについてご紹介します。

HTML/CSS 参考書の指示に沿ったコーデイングから始まり、講師が過去に制作したサイトのデザインデータの模写を行いました。
課外活動として、講師が依頼を受けたサイト制作の補助(一部ページのコーディングを担当)も行っております。
ポートフォリオ作成に当たり、個人で制作したwebサイトもございます。
JavaScript 動的webサイト作成のためのプログラミングや、jqueryの活用が可能です。
PHP/WordPress htmlファイルで作成したwebサイトをWordPress化し活用することができます。
PhotoShop/Illustrator デザインデータの確認やwebサイト作成の際の画像のアセット作成の他、画像編集やロゴ作成などのデザイン作成に活用することができます。
クラウドソーシングにてバナーデザイン等のコンペティションにも参加しております。
